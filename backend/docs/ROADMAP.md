# TrustRAG Roadmap

Phases:

1. Project skeleton (scaffolding)
2. Backend core: FastAPI, auth, users, docs
3. Frontend core: React + TS app skeleton
4. Document ingestion pipeline
5. Embedding service and Qdrant integration
6. Adaptive retrieval module (BM25 / dense / hybrid)
7. LangGraph agents orchestration
8. Evidence verification & explainability
9. Report generation
10. Evaluation and experiments
11. Paper writing

Use incremental development: implement and test each phase independently.
